HTML: El lenguaje de marcado de hipertexto

¿Cuál es la utilidad de HTML? ¿Qué conjunto mínimo de tags debe contener un documento elaborado en este lenguaje? Descripción breve de su utilidad.

El lenguaje HTML permite la creación de documentos basados en hipertexto con una estructura organizada semánticamente y estructuralmente organizada. Además posibilita el uso de contenido multimedia como imágenes, sonido y video, así como de programas ejecutables (scripts) incrustados en los documentos. Una de las características más notable de HTML es la posibilidad de navegar a través de hipervínculos, es decir, enlaces a otros documentos.

Los elementos que todo documento HTML debe contener son:

<!DOCTYPE HTML>

DOCTYPE no es realmente un tag, sino que es un preámbulo cuya función es indicar que el lenguaje del documento es HTML.

<html>

El tag html se utiliza para indicar el elemento raíz del documento, lo que significa que todos los demás elementos están contenidos en él.

<head>

El tag head se utiliza para indicar la sección de metadatos del documento. Este elemento puede contener enlaces a recursos relacionados como hojas de estilo y scripts.

<title>

El tag title se utiliza para indicar el título del documento. Debe estar dentro de la sección head.

<body>

El tag body se utiliza para indicar la sección de datos del documento. Este elemento puede contener elementos visibles como texto, imagenes y contenido multimedia.
